# Proyecto Ejercicio Karate

Este proyecto utiliza Gradle 7.6, Java 1.8, Karate, y JUnit 5 para realizar pruebas de API en Swagger Petstore.

## Prerequisitos

Asegúrate de tener instalado:

- [Java JDK 1.8](https://www.oracle.com/java/technologies/javase-downloads.html)
- [Gradle 7.6](https://gradle.org/install/)
- [IntelliJ IDEA 2023.1](https://www.jetbrains.com/idea/download/)
- Variables de entorno configuradas para Java y Gradle.

## Comandos de Instalación

 git clone https://github.com/23456-Tony-Larrea/NTTDATA-API-QA-CHALLENGE
 cd NTTDATA-API-QA-CHALLENGE



Instrucciones para Ejecutar los Test
Configuración del Proyecto en IntelliJ:

Abre IntelliJ IDEA y selecciona "Import Project".
Navega hasta la carpeta de tu proyecto y selecciona el archivo build.gradle.
Sigue las instrucciones para importar el proyecto.
Ejecución de Pruebas en IntelliJ:

Abre la ventana "Gradle" en el lado derecho.
Navega hasta Tasks -> verification -> test.
Haz doble clic en test para ejecutar las pruebas.
Ver Reportes:

Puedes encontrar los reportes de Karate en build/karate-reports.
